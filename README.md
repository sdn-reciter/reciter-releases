# Reciter

Android-приложение для чтения электронных книг с озвучиванием голосом.

[**→ Скачать последнюю версию**](../../releases/latest)

## Возможности

- **Чтение книг** в форматах FB2 и EPUB (в том числе упакованные в `.zip`).
- **Озвучивание текста** через системный голос Android с настройками скорости, тона, голоса и пауз между абзацами. Подсветка читаемого слова, пауза при сворачивании, продолжение с того же места при возврате.
- **Библиотека** с сериями, полками, закладками и оглавлением. Сортировка, поиск, фильтры по статусу прочтения.
- **ИИ-разбор глав и абзацев** — улучшает структуру плохо размеченных книг (особенно полезно для PDF).
- **Переводчик книг** ru↔en — переводит весь текст книги офлайн на устройстве.
- **Облачная синхронизация** прогресса и библиотеки через Google Drive или Яндекс.Диск, с раздельным хранением данных по аккаунту.
- **Две дизайн-системы** на выбор: Classic (мягкие формы) и Brutalist (моноширинный шрифт, острые углы).
- **Темы оформления**: Бирюза, Сепия, Циан, Графит, Рубин + пользовательские цвета.
- **Правила произношения** — собственный словарь для замены слов, ударений, ё/е.
- **Таймер сна**, автояркость, ручная регулировка размера шрифта, межстрочного интервала и отступов.
- **Мини-игры в читалке** во время прослушивания: словесная змейка, лопни букву и другие.

## Установка

1. Скачайте `.apk` со страницы [Releases](../../releases/latest).
2. Откройте файл на телефоне — Android запросит разрешение установить из неизвестного источника. Разрешите однократно для браузера или файлового менеджера.
3. Готово. Дальнейшие обновления приложение скачивает само (Настройки → «Проверить обновления»).

**Требования:** Android 7.0 или новее, свободного места — около 150 МБ.
