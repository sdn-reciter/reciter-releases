# Словари ударений — скрипты сборки

Скрипты, которыми собираются офлайн-словари ударений для приложения. Как
словарь работает в читалке, форматы меток и лицензии — в
[`../accent-dictionary.md`](../accent-dictionary.md).

| Скрипт | Собирает | Ассет |
|---|---|---|
| [`make_dict_wiktionary.py`](make_dict_wiktionary.py) | Русский Викисловарь (kaikki.org), ~887 тыс. форм, CC BY-SA | `app/src/main/assets/ruaccent_dict_wiktionary.dict` |
| [`make_dict_ruaccent.py`](make_dict_ruaccent.py) | RUAccent, ~3.2 млн форм, CC BY-NC-ND (только dev) | `app/src/dev/assets/ruaccent_dict_ruaccent.dict` |

## Формат

Строки `ключ\tпозиция` (позиция ударной гласной), **отсортированы по байтам**
для бинарного поиска по memory-map, затем `gzip`. Приложение разжимает файл,
mmap'ит и ищет слово; `U+0301` вставляет само.

> Расширение `.dict`, а не `.gz`: AAPT2 молча распаковывает ассеты с `.gz` при
> упаковке APK и снимает суффикс — приложение не находило файл. `.dict` AAPT
> не трогает.

## Пересборка

```bash
# Викисловарь (публичный): качает дамп kaikki (~770 МБ)
python make_dict_wiktionary.py
# при 404 на дамп — возьми свежую ссылку с kaikki.org/dictionary/Russian/ и --url <ссылка>

# RUAccent (только личное/dev)
pip install ruaccent && python make_dict_ruaccent.py
```

Готовый файл заменить в соответствующей папке ассетов. В Colab скрипты сами
запускают скачивание; если не стартовало — печатают, как забрать вручную.

Покрытие RUAccent больше (3.2 млн против 887 тыс. форм), но он под NoDerivatives
и в публичную сборку не идёт. Омографы (за́мок/замо́к) без нейромодели ставятся
«по умолчанию» — максимум качества даёт режим «Сервер» (`/accent`, см.
[`../server-api.md`](../server-api.md) §3).
